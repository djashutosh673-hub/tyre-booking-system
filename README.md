# Tyre Puncture Booking System

A complete Uber‑style booking system with real‑time tracking using Node.js, Express, Socket.io, Google Maps, and a JSON file database.

## Features
- User: select location on map, book puncture/tyre service, live track mechanic.
- Mechanic: see incoming requests, accept, send live location (GPS).
- Real‑time updates via WebSockets (Socket.io).
